The purpose of this directory is to get you started with the Tkinter toolkit.

Tkinter:
Tkinter is a Python binding to the Tk GUI toolkit.
Tk is the original GUI library for the Tcl language.
Tkinter is implemented as a Python wrapper around a complete Tcl interpreter embedded int the python interpreter.
There are several other popular Python GUI toolkits.
Most popular are wxPython, PyQt, and PyGTK.
